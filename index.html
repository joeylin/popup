Skin

Skins are a combination of Javascript Options, CSS and a PNG sprite. To use a skin the only option required is the skin option. How to use Skin, fllow below steps:

Javascript

use Javascript to Popup.skins set for creating a new skin


$.extend($.Popup.skins, {
  'skinRimless' : {
    ui: 'inside'
  }
});

After the skin has been created it can be applied using the skin option:


<a class="popup" href="img/1.jpg" data-popup-skin="skinRimless"><img src="img/1-thumbnail.jpg" alt="" /></a>

When using groups the best way to apply the skin is using the data-popup-group-options:


<a class="popup" href="img/1.jpg" data-popup-group="gallery1" data-Popup-group-options="skin:'skinSimple'"><img src="img/1-thumbnail.jpg" alt="" /></a>
<a class="popup" href="img/2.jpg" data-popup-group="gallery1" ><img src="img/2-thumbnail.jpg" alt="" /></a>
<a class="popup" href="img/3.jpg" data-popup-group="gallery1" ><img src="img/3-thumbnail.jpg" alt="" /></a>

CSS

Once the Javascript for a skin is in place a skin will need CSS and a sprite for styling

We have a basic css style for every component, fllowing below to create the new style


.skinSimple .popup-thumbnails-holder {
    //here you can override and inherit .popup-thumbnails-holder
    //and this style if only for wihteBorder skin
}

click here to check a list of classnames that become available

Set Default Skin

To avoid having to set the skin option all the time the default skin can be changed using $.Popup.setDefaultSkin(). It should be called after the file that defines skinRimless skins, for example:


<script type='text/javascript'>
  // make a skinRimless skin the new default
  $.Popup.setDefaultSkin('skinRimless');
</script>;

API Method

The alternative to using the HTML based approach is using the Javascript API. The API has all the options of the HTML approach and provides some useful extra functions.

show

    url: 'img/1.jpg'
},{
    url: 'img/2.jpg',
    title: 'captions'
}],
{
    height: '400px',
    width: '400px',

    skin: 'skinSimple',
    transition: 'dropdown'
});
click to show popup 

$.Popup.show(['img/1.jpg', 'img/2.jpg'], {
    height: '400px',
    width: '400px',
    fullscreen: false
});

getCurrent

This method is used to fetch the current HTML element that Popup is associated with. Returns a jQuery object containing the element.


$.Popup.getCurrent()

$.Popup.show(['img/1.jpg', 'img/2.jpg', 'img/3.jpg'], 2);
setTimeout(function(){
    alert($.Popup.getCurrent());
    $.Popup.next();
    alert($.Popup.getCurrent());
    $.Popup.close();
},3000);
click to get current

hasNext

Returns true if there is another item to display in the current gallery. Note that this will always be true in galleries that are continuous.


$.Popup.hasNext()

$.Popup.show(['img/1.jpg', 'img/2.jpg', 'img/3.jpg'], 2);
setTimeout(function(){
    alert($.Popup.hasNext());
    $.Popup.next();
    alert($.Popup.hasNext());
    $.Popup.close();
},3000);
