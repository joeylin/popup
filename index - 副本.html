参考 https://github.com/mnelson/diabox/blob/master/Source/diabox.js 的 register_renderable 实现


<a href="http://www.samplepdf.com/sample.pdf" class="type-gdoc">Open with Google doc</a>
<script>
//gdoc extend remote(iframe) type
Popup.RegisterType('gdoc', {
    extends: 'iframe',
    initialize : function(options){
        options.url = "http://docs.google.com/viewer?embedded=true&url=" + options.url;
    }
});
jQuery(document).ready(function(){
    $('.type-gdoc').Popup(
        {type: 'gdoc',title:'This is Google Docs.'}
    );
});
</script>

registerComponent


<a class="api-skin-small" href="img/1.jpg">use the new register skin</a>
<script>
jQuery(document).ready(function(){
    $.Popup.registerComponent('rotate',{
        html: "<div class="rotate"><span class="left"></span><span class='right'></span></div>",
        func: function() {
            $('.rotate').find('.left').on('click',function() {
                $.Popup.rotate(-90);
            });
            $('.rotate').find('.right').on('click',function() {
                $.Popup.rotate(90);
            });
        }
    });
});
</script>

The alternative to using the HTML based approach is using the Javascript API. The API has all the options of the HTML approach and provides some useful extra functions.

destory

Removes all traces of Popup from the document. Not the same as Popup.close(), which tucks popup away for future use.


Popup.destory();
click to destory

isOpened

Returns true if the popup is opened.


alert(Popup.isOpened());
click to check if opened

current

This method is used to fetch the current HTML element that Popup is associated with. Returns a jQuery object containing the element.


var $current = Popup.current();
alert($current.html());
click to get current

hasNext

Returns true if there is another item to display in the current gallery. Note that this will always be true in galleries that are continuous.


alert(Popup.hasNext());
click to check if hasNext

play

Resumes the current slideshow (if there is one).


$.Popup.play();

pause

Pauses the current slideshow (if there is one).


$.Popup.pause();
click to pause slideshow

isPaused

Returns true if the popup is currently in a paused state. This can happen if pause is invoked in the middle of a slideshow.


alert(Popup.isPaused());
